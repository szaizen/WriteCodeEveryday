九九
以下のような表記で、九九を出力して終了するプログラムを作成してください。×記号の代わりに、小文字の x を使用してください。

1x1=1
1x2=2
.
.
9x8=72
9x9=81

function problem1() {
	for( var i = 1; i <= 9; i++ ) {
		for ( var j = 1; j <= 9; j++ ) {
			console.log(i + "x" + j + "=" + i * j);
		}
	}
};

問題2

山の高さ
山の高さをメートル単位の整数で表した 10 個のデータがあります。
その 10 個のデータを読み込み、その中で、高い順から３つ出力するプログラムを作成して下さい。
0 ≤ 山の高さ ≤ 10,000

function problem2() {

	/* randomな10の数を生成 */
	var mountainHeight = [];
	for( var i = 0; i < 10; i++ ) {
		mountainHeight.push(Math.ceil( Math.random()*10000 ));
	};

	/* 降順で並び替え */
	mountainHeight.sort(function(a,b){
	        if( a > b ) return -1;
	        if( a < b ) return 1;
	        return 0;
	});

	$('.answer2').append("<div>" + mountainHeight[0] + "</div>").append("<div>" + mountainHeight[1] + "</div>").append("<div>" + mountainHeight[2] + "</div>");

};
